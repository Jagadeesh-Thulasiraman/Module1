## ðŸ§  Algorithm
1. Get an input from the user.
2. Convert the input to an integer and store it in a variable `a`.
3. Use the modulo operator `%` to check if `a % 2 == 0`.

   - Else, print `"ODD"`.
4. End the program.

## ðŸ§¾ Program
def check_even_odd(number):
    if number % 2 == 0:
        return "EVEN"
    else:
        return "ODD"

number = int(input()) 

result = check_even_odd(number)
print(result)
